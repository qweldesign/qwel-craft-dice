# ダイスの回転シミュレーター | Dice Roll Simulator

4方向のボタンを押すと、ダイスがその方向に転がって出目を示す  
canvasを使った2Dの描画なので、回転アニメーションまでは実装できず...  
それから、できれば立方体を角丸にしてあげたかった...
